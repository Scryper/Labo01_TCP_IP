5 fonctionnalités principales:

	1) IP -> déterminer la classe, le nombre de réseaux et le nombre d'hotes
	2) IP + masque -> adresse réseau et adresse de broadcast du réseau.
	3) IP + masque + adresse réseau -> est ce que l'ip appartient au SR
	4) IP machine + masque machine + adresse réseau -> IP est une machine du réseau?
	5) 2 IP + 2 masques -> même réseau?
